Getting Started
  • Plugin Help Section
  • How Plugins work
  • How to create your plugins
  • 1. Duplicate the folder C:\Arc\plugins\template-page
  • 2. Rename the folder my-new-plugin
  • 3. Open plugins\my-new-plugin\plugin.js file
  • 4. To activate the plugin, you have to uncomment the first part by removing /* line 4 and */ line 19
  • 5. Disable cache, when working on plugins you need to disable the your browser cache to see the changes
  • 6. Refresh Arc, you should now see your new plugin under tools in the left-navigation pane with name Template Title
  • 7.To update the menu title, replace Template Title line 8 of the plugin.js with your new title
  • 8.To update the menu icon, replace fa-paw line 9 of the plugin.js with your new icon
  • 9.Make your plugin unique by replacing arcTemplate line 8 and 21
  • 10. Replace the folder name template-page line 28 with your new folder name

      $tm1.instances().then(function (data) {
         $scope.lists.instances = data;
         //Open Modal

      });
   };

Get information of a specific instance:
x 
 plugin.js:
   $scope.getInstance = function (name) {
      $scope.options.showInstanceInfo = !$scope.options.showInstanceInfo;
      $tm1.instance(name).then(function (data) {
         $scope.lists.instanceData = data;
         $scope.openModalInstances($scope.selections.instance, $scope.lists.instanceData);
      });
   };
 template.html:
ng-click="getInstance(selections.instance)"

Modals and popups

x 
   
    plugin.js
   $scope.openModal = function (title, message, size) {
      var dialog = ngDialog.open({
         className: "ngdialog-theme-default "+size,
         template: "__/plugins/cheatsheet/modal.html",
         name: "Instances",
         controller: ['$rootScope', '$scope', function ($rootScope, $scope) {

            $scope.title = $scope.ngDialogData.title;
            $scope.message = $scope.ngDialogData.message;

         }],
         data: {
            title: title,
            message: message
         }
      });
   };
    template.html:
   ng-click="openModal('My small modal','My message','small')"
